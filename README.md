# Foosball Arcade

Foosball Arcade is a browser-based arcade foosball game built as a static HTML/CSS/JavaScript project.

## Run Locally

Open `index.html` through any static web server. The game uses ES modules, so direct `file://` loading is not recommended.

## GitHub Pages

The project uses relative asset paths and does not require a build step, so it can be hosted directly from a GitHub Pages branch or folder.

## First Playable Notes

- Human player controls the red team on the left.
- Blue team on the right is AI controlled.
- The match is first to 8 points.
- Gameplay is designed for landscape touch screens.
- The menu supports mouse or touch.
- Ball and impact audio are generated with Web Audio synthesis.
